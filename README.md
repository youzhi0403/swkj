# swkj
生物科技公司门户网站管理项目，分为两个部分，一个是公司的门户网站，一个是门户网站的管理系统。

项目地址：待部署

项目介绍

整个项目包括两个部分，一个是公司的门户网站，包括对主页，产品，热销，新闻，联系我们等信息的展示。一个是门户网站的管理系统，包括对主页，产品，热销，新闻，联系我们等信息的管理。目前展示模块，管理模块已经开发完成；权限模块还在开发当中。
